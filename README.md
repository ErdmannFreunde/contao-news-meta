# Contao News Meta

Fügt zu den Contao News die Felder **Meta-Title** und **Meta-Description** hinzu und gibt diese auf der News-Detail-Seite aus.
Werden die Felder nicht ausgefüllt, wird der Contao Standard verwendet.
